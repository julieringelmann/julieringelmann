Africa Connect: Rep Training App
A tablet app for the training and development of Reckitt Benckiser representatives

Description

  • Client: Reckitt Benckiser
  • ECD: Simon Gill

The brief:
To design a simple app that allows for the updating, training and testing of Reckitt Benckiser drug reps.

The solution:
An interactive application with the ability to track user progress structured around their interaction with 4 main content sections: the user panel, the guide library, the training modules, and the certificates.

The user panel provides the user with a complete profile overview, consisting of notifications, user completion averages, and user pass/fail averages.

The detailing guide library is periodically updated, with each new arrival being announced in the notifications panel. Every detailing guide is accompanied by a corresponding test that is added to the training library.

Each test has an expiration date, before which the rep should have completed it. All tests; passed, in progress, not yet started, and tests expired; are tallied and displayed in the user panel to be tracked by the user and their respective managers.

Overall the project was deemed a success as we were able to deliver an effective, beautiful and fully functional application. Phase two of this design project intended to explore the inclusion of a full user dashboard. In this case the user could be able to compare and compete with other reps, and earn badges, points and rewards for the best scores.
